# Branding Guide

How to write BRANDING.md — the identity document for a project's public face.
Optional but valuable for any project with users, contributors, or a public presence.

## Principles

1. **Serve the product.** Every visual choice makes the project clearer and more memorable.
2. **Consistency over perfection.** Simple rules, consistently applied.
3. **Match the audience.** Developer tools ≠ consumer apps ≠ enterprise products.
4. **Less is more.** 2-3 colors, 1-2 fonts, one strong logo concept.

## Template

### 1. Name & Identity
- **Name**: [name]
- **Pronunciation**: [if not obvious]
- **In code**: `[lowercase-kebab-case]`
- **Tagline**: [5-10 words capturing the essence]

### 2. Logo
- Concept, visual metaphor, connection to project's purpose
- Primary mark, icon mark (small sizes), wordmark (text-only)
- Minimum size, clear space requirements

### 3. Color Palette
- Brand colors (primary, secondary, accent) with Hex, RGB, usage
- Neutrals (text, background, surface, border)
- Semantic colors (success, error, warning, info)
- Dark mode adjustments

### 4. Typography
- Font stack for headings, body, code
- Type scale: H1-H3, body, small, code
- Line heights and weights

### 5. Voice & Tone
- 3-5 personality adjectives with behavioral descriptions
- Writing rules for headlines, documentation, error messages, marketing

### 6. Visual Language
- Border radius (consistent values: 4px, 8px, 12px)
- Shadows / elevation levels
- Spacing scale (base 4px or 8px)
- Icon library (Lucide, Heroicons, Phosphor, Tabler)

### 7. Assets Checklist
- Logo (full, SVG + PNG)
- Icon (SVG + PNG, multiple sizes)
- Favicon (.ico + .png)
- OG Image (1200×630)
- Social Banner (1500×500)

## Branding Depth by Project Type

| Project Type | Sections | Depth |
|-------------|---------|-------|
| CLI / System Tool | Name, Logo, Color (terminal), Voice | Light |
| Library / SDK | Name, Logo, README aesthetics, Docs style | Light |
| Open Source Tool | Name, Logo, Color, Voice, Social, README | Medium |
| SaaS / Web App | All sections | Full |
| Internal Tool | Name, Color, Typography | Minimal |
